About Me

สวัสดีครับ ผมชื่อ นายไพศาล อิงควระ เรียนจบมาจากมหาวิทยาลัยกรุงเทพ คณะเทคโนโลยีสารสนเทศและนวัตกรรม สาขาวิชาเกมและสื่อเชิงโต้ตอบ สนใจในตำแหน่งงาน Game Programmer ผมได้มีโอกาสไปฝึกงานกับบริษัท Urnique Studio ในระยะ 6 เดือนและ
มีความสนใจในด้าน Graphic กำลังศึกษา Shader ใน Unity และสามารถนำ Tool มาปรับใช้กับโปรเจคเพื่อลดเวลาทำงานได้

Recent Work

Place of Destination

Place of Destination เป็นเกมแนว 2D Action Roguelike ที่เราต้องเอาชนะมอนสเตอร์ในแต่ละรอบและอัปเกรดตัวละครให้เก่งขึ้นด้วยการซื้อ item และยังมีระบบการจีบตัวละครอีกด้วย จุดเด่นของเกมนี้คือการที่เราเห็นตัวละครที่เล่นมีการมีพัฒนาการที่ชัดเจนเช่นการโจมตีที่แรงขึ้น ในเกมนี้มีคนทำทั้งหมด 4 คน ผมรับหน้าที่เป็นโปรแกรมเมอร์และ Project Lead ในงานนี้เป็นคนทำ Core Gameplay การใส่ Animation ให้ตัวละครต่างๆ และระบบ Dialogue ที่ได้ใช้ตัว Ink เข้ามาช่วยในการทำ

Internship Project

โปรเจกต์ที่ได้ทำร่วมกับทาง Urnique Studio ในตำแหน่ง Game Programmer แต่ไม่สามารถลงเกี่ยวกับรายละเอียดของโปรเจกต์ได้ และของที่ผมได้ทำส่วนใหญ่เกี่ยวกับ UI ที่คล้ายๆ กับ Papers, Please การใช้ Tool จาก Asset store เอามาปรับใช้ให้เข้ากับเกมแบบที่ทางทีมต้องการ และการใช้ Json ในการนำเข้าข้อมูลต่างๆ ภายในเกมสามารถลดเวลาได้จาก 1-2 ชั่วโมงเหลือแค่ 2 นาที

Let's Go My Car

Let's Go My Car เป็นเกม Racing Multiplayer ที่ผู้เล่นสามารถเล่นกับเพื่อนได้ถึง 4 คนผ่านระบบ Steam เกมนี้ได้ใช้ตัว API จาก Mirror และ Steamworks เข้ามาช่วยในการทำระบบ Multiplayer ในเกมนี้ได้รับหน้าที่เป็นโปรแกรมเมอร์ และได้ทำระบบที่เกี่ยวกับ Multiplayer ทั้งหมดเช่นระบบล็อบบี้และให้ภาพทุกคนเห็นเหมือนกันและยังทำตัว Core Gameplay อีกด้วย

Train To Hell - VR

Train To Hell เป็นเกมที่พัฒนาให้กับเครื่อง VR เกมนี้จะได้รับแรงบันดาลใจมาจากเกมตู้ 4D ที่ให้เรานั่งรถไฟออกไปผจญภัยไปเรื่อยๆ และมี Enemy ให้ผู้เล่นสามารถยิงเพื่อเก็บคะแนนได้ โดยเกมนี้ผมได้รับหน้าที่เป็นโปรแกรมเมอร์ที่เขียนระบบต่างๆ ภายในเกมเช่น ระบบปืน ระบบเสียง เป็นต้นและความยากในงานนี้คือการใช้ต้องมาเขียนระบบต่างๆ ให้ใช้กับ VR ได้
