About Me

สวัสดีครับ ผมชื่อ นายไพศาล อิงควระ เรียนจบมาจากมหาวิทยาลัยกรุงเทพ คณะเทคโนโลยีสารสนเทศและนวัตกรรม สาขาวิชาเกมและสื่อเชิงโต้ตอบ สนใจในตำแหน่งงาน Game Programmer ผมได้มีโอกาสไปฝึกงานกับบริษัท Urnique Studio ในระยะ 6 เดือน และพร้อมที่จะเรียนรู้สิ่งใหม่ๆ ตลอดเวลา มีความสามารถที่จะหาความรู้และแก้ปัญหาได้ด้วยตัวเอง

Recent Work

Place of Destination

Place of Destination เป็นเกมแนว 2D Action Roguelike ที่เราต้องเอาชนะมอนสเตอร์ในแต่ละรอบและอัปเกรดตัวละครให้เก่งขึ้นด้วยการซื้อ item และยังมีระบบการจีบตัวละครอีกด้วย จุดเด่นของเกมนี้คือการที่เราเห็นตัวละครที่เล่นมีการมีพัฒนาการที่ชัดเจนเช่นการโจมตีที่แรงขึ้น ในเกมนี้มีคนทำทั้งหมด 4 คน ผมรับหน้าที่เป็นโปรแกรมเมอร์และ Project Lead ในงานนี้เป็นคนทำ Core Gameplay การใส่ Animation ให้ตัวละครต่างๆ และระบบ Dialogue ที่ได้ใช้ตัว Ink เข้ามาช่วยในการทำ

Let's Go My Car

Let's Go My Car เป็นเกม Racing Multiplayer ที่ผู้เล่นสามารถเล่นกับเพื่อนได้ถึง 4 คนผ่านระบบ Steam เกมนี้ได้ใช้ตัว API จาก Mirror และ Steamworks เข้ามาช่วยในการทำระบบ Multiplayer ในเกมนี้ได้รับหน้าที่เป็นโปรแกรมเมอร์ และได้ทำระบบที่เกี่ยวกับ Multiplayer ทั้งหมดเช่นระบบล็อบบี้และให้ภาพทุกคนเห็นเหมือนกันและยังทำตัว Core Gameplay อีกด้วย

BonkSlayer

BonkSlayer เป็นเกมแนว 2D Top-Down Shooter และยังเป็นเกมแรกที่ได้ทำขึ้นใน Unity ตัวละครในเกมนี้ได้รับแรงบันดาลใจสุนัขพันธุ์ชิบะที่เป็น meme bonk เกมนี้ผู้เล่นได้สู้กับสุนัขประเภทต่าง ๆ โดยใช้ baseball bat และเอาชีวิตรอดให้ได้นานที่สุด ได้รับหน้าเป็นโปรแกรมเมอร์และได้ทำระบบทุกอย่างภายในเกม

LifeForceTenka Clone

LifeForceTenka Clone เป็นเกมแนว FPS ที่จะให้ผู้เล่นได้เล่นผ่านด่านและต้องเอาชนะอุปสรรคต่างๆ เกมนี้ทำด้วย Unreal Engine 4 และได้ใช้ Blueprint ในการเขียนระบบต่างๆ ภายในเกมและเกมนี้เป็นเกมที่ได้ทำตามคอร์สและเอา Asset ทั้งหมดมาจาก Unreal Engine 4 FPS:Lifeforce Tenka Clone course.
